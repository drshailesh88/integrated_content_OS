# Carousel Design Standards

Research-backed design guidelines for maximum engagement.

---

## Engagement Research

### Aspect Ratio Impact
| Ratio | Engagement | Best For |
|-------|------------|----------|
| 4:5 (1080×1350) | +10% vs square | Instagram primary |
| 1:1 (1080×1080) | Baseline | Multi-platform |
| 16:9 (1920×1080) | -15% on mobile | Avoid for carousels |

**Source:** HubSpot Social Media Report 2024

### Slide Count
| Count | Avg. Engagement | Notes |
|-------|-----------------|-------|
| 3-4 | 2.1% | Too short, low value |
| 5-7 | 2.5% | Good for simple topics |
| **8-10** | **2.9%** | Optimal for education |
| 11+ | 2.2% | Fatigue sets in |

**Source:** Later.com Instagram Carousel Study

### First Slide Impact
- 80% of engagement determined by hook quality
- Users decide in 0.3 seconds whether to swipe
- Questions increase swipe rate by 23%
- Numbers in titles increase swipe rate by 18%

## Typography

### Mobile-First Sizing
| Element | Min Size | Recommended | Max Size |
|---------|----------|-------------|----------|
| Headlines | 36px | 48px | 72px |
| Body | 24px | 28px | 32px |
| Captions | 18px | 22px | 26px |
| Statistics | 64px | 72px | 96px |

### Line Length
- Headlines: 2-3 lines maximum
- Body text: 3-4 lines maximum
- Single tip: 1-2 lines ideal

### Font Weights
| Use Case | Weight |
|----------|--------|
| Headlines | Bold (700) |
| Subheadlines | SemiBold (600) |
| Body | Regular (400) |
| Captions | Medium (500) |
| Statistics | Bold (700) or ExtraBold (800) |

## Color & Contrast

### WCAG AA Requirements
| Text Type | Min Contrast | Our Design |
|-----------|--------------|------------|
| Normal text | 4.5:1 | 7.5:1 (white on primary) |
| Large text (18pt+) | 3:1 | 7.5:1 |
| UI components | 3:1 | 4.5:1+ |

### Color Mode Selection
| Slide Type | Mode | Rationale |
|------------|------|-----------|
| Hook | Dark | Bold, attention-grabbing |
| Tips | Light | Easy to read, scannable |
| Stats | Light | Numbers stand out |
| Myth | Light | Clear myth/truth contrast |
| Quote | Dark | Gravitas, authority |
| CTA | Dark | Action-oriented |

### Accent Color Usage
- Bullets and icons: Primary accent (coral)
- Highlights: Sparingly, max 2 per slide
- Numbers: Primary color (teal) for emphasis

## Spacing & Layout

### Margins & Padding
| Element | Value | Purpose |
|---------|-------|---------|
| Outer margin | 80px | Breathing room |
| Content padding | 40px | Internal spacing |
| Gap (small) | 16px | Between related items |
| Gap (medium) | 24px | Between sections |
| Gap (large) | 40px | Major divisions |

### Footer Specifications
| Element | Value |
|---------|-------|
| Height | 150px |
| Photo size | 100×100px (circular) |
| Separator | 2px, secondary color |

### Visual Hierarchy
1. **Primary**: Headline or statistic (largest)
2. **Secondary**: Supporting text or subtitle
3. **Tertiary**: Body text or details
4. **Quaternary**: Footer and captions

## Text Density

### Words Per Slide
| Content Type | Max Words | Notes |
|--------------|-----------|-------|
| Hook | 8-12 | Punchy, curiosity-driven |
| Tips | 12-15 | Scannable bullets |
| Stats | 5-8 | Number + context |
| Myth | 10-15 | Myth + truth |
| Quote | 15-25 | Allow longer quotes |
| CTA | 4-8 | Action + handle |

### Bullet Points
- Maximum 3-4 bullets per slide
- Each bullet: 5-10 words
- Use icons for visual appeal

## Icon Guidelines

### Style Selection
| Context | Style | Example |
|---------|-------|---------|
| Decorative | Outline | Background icons |
| Key points | Filled | Bullet icons |
| Actions | Filled | CTA icons |

### Size Guidelines
| Use | Size |
|-----|------|
| Inline | 24-32px |
| Bullet | 48px |
| Featured | 64px |
| Hero | 96px+ |

### Color Rules
- Match icon color to text or accent
- Dark mode: white or accent icons
- Light mode: primary or accent icons

## Slide Type Specifications

### Hook Slide
- Dark background (primary color)
- Centered title, large font (48-72px)
- Optional subtitle below
- No footer (clean, focused)
- Question mark or number preferred

### Tips Slide
- Light background
- Left-aligned title
- Bullet points with accent icons
- Footer with branding
- 3 bullets maximum

### Stats Slide
- Light background
- Centered statistic (72px+)
- Label below number
- Context text (smaller)
- Footer with branding

### Myth Slide
- Light background
- Clear "MYTH" label (red)
- Strikethrough on myth text
- "TRUTH" label (teal)
- Separator line between sections

### Quote Slide
- Dark background
- Large opening quote mark
- Quote text (32px, white)
- Author attribution
- No footer (clean)

### CTA Slide
- Dark background
- Centered action text
- Handle below in accent color
- Optional profile photo
- Clean, minimal design

---

## Anti-Patterns

### Avoid These
- Too much text (>20 words)
- Clashing colors
- Inconsistent margins
- Mixed font families
- Low contrast text
- Pixelated icons
- Busy backgrounds
- Multiple CTAs per slide

### Red Flags
- AI-generated phrases
- Em dash overuse
- Weak language ("might", "could")
- Clickbait without substance
- Generic stock imagery

---

## Quality Checklist

### Before Publishing
- [ ] Hook is compelling (question or number)
- [ ] 8-10 slides total
- [ ] ≤15 words per slide
- [ ] Contrast ratio ≥4.5:1
- [ ] No AI-detected patterns
- [ ] CTA on last slide
- [ ] Branding consistent
- [ ] Mobile preview checked
